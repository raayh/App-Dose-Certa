# **🟡 PREPARAÇÃO TÉCNICA**

Este guia é para ser seguido **do zero** no repositório da faculdade. Siga cada passo exatamente como descrito, copiando os comandos e os códigos para os arquivos indicados.

## 🚀 Passo 1: Inicialização do Projeto Expo

No terminal, dentro da pasta do projeto da faculdade:

> [!WARNING]
Como a pasta já pode conter arquivos (como o `.git` ou a proposta inicial), o comando abaixo pode perguntar se você deseja continuar. **Digite "Yes" (ou pressione Enter)**. Ele vai baixar o template por cima, mantendo o controle do Git.
> 

```bash
npx create-expo-app@latest ./ --template tabs
```

Após a criação, salve o primeiro commit:

```bash
git add .
git commit -m "chore: setup expo environment and merge main files"
```

## 🔥 Passo 3: Configuração do Firebase e Login Guard

Agora vamos criar a inteligência de autenticação.

### 1. Criar pasta e arquivo de config

Crie a pasta `services` na raiz e o arquivo `firebaseConfig.ts`:

**Arquivo:** `services/firebaseConfig.ts`

```tsx
import { initializeApp } from "firebase/app";
import { getAuth } from "firebase/auth";
import { getFirestore } from "firebase/firestore";

const firebaseConfig = {
  apiKey: process.env.EXPO_PUBLIC_FIREBASE_API_KEY,
  authDomain: process.env.EXPO_PUBLIC_FIREBASE_AUTH_DOMAIN,
  projectId: process.env.EXPO_PUBLIC_FIREBASE_PROJECT_ID,
  storageBucket: process.env.EXPO_PUBLIC_FIREBASE_STORAGE_BUCKET,
  messagingSenderId: process.env.EXPO_PUBLIC_FIREBASE_MESSAGING_SENDER_ID,
  appId: process.env.EXPO_PUBLIC_FIREBASE_APP_ID,
};

const app = initializeApp(firebaseConfig);
export const auth = getAuth(app);
export const db = getFirestore(app);
```

### 2. Implementar o Auth Guard

Abra o arquivo `app/_layout.tsx` e **substitua todo o conteúdo** por este (que contém a lógica de desviar usuários deslogados para o login):

**Arquivo:** `app/_layout.tsx`

```tsx
import { DarkTheme, DefaultTheme, ThemeProvider } from '@react-navigation/native';
import { Stack, useRouter, useSegments } from 'expo-router';
import { StatusBar } from 'expo-status-bar';
import { useState, useEffect } from 'react';
import { onAuthStateChanged } from 'firebase/auth';
import { auth } from '@/services/firebaseConfig';
import { useColorScheme } from '@/hooks/use-color-scheme';

export default function RootLayout() {
  const colorScheme = useColorScheme();
  const [user, setUser] = useState<any>(null);
  const [initializing, setInitializing] = useState(true);
  const router = useRouter();
  const segments = useSegments();

  useEffect(() => {
    const subscriber = onAuthStateChanged(auth, (user) => {
      setUser(user);
      if (initializing) setInitializing(false);
    });
    return subscriber;
  }, []);

  useEffect(() => {
    if (initializing) return;

    const inAuthGroup = segments[0] === '(auth)';

    if (!user && !inAuthGroup) {
      router.replace('/(auth)/login');
    } else if (user && inAuthGroup) {
      router.replace('/(tabs)');
    }
  }, [user, segments, initializing]);

  if (initializing) return null;

  return (
    <ThemeProvider value={colorScheme === 'dark' ? DarkTheme : DefaultTheme}>
      <Stack>
        <Stack.Screen name="(tabs)" options={{ headerShown: false }} />
        <Stack.Screen name="(auth)" options={{ headerShown: false }} />
      </Stack>
      <StatusBar style="auto" />
    </ThemeProvider>
  );
}
```

