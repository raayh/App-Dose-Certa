# **🟢 PREPARAÇÃO (Semanas 1 e 2)**

Este guia engloba as diretrizes fundamentais iniciais do projeto. O objetivo das duas primeiras semanas foi construir a base idealizada e os alicerces teóricos do aplicativo **Dose Certa**.

---

## 🏗️ Passo 1: Estrutura Base e Repositório

Foi determinado que o projeto obrigatoriamente iniciaria com o versionamento profissional.

1. **Github:** Criação do repositório remoto para o grupo.
2. **Setup Git Local:** Conexão através do `git remote add`.
3. **Documentação Inicial:** Criação do arquivo de "Contrato" (a nossa [proposta.md](../../proposta.md) oficial) que ditaria todas as métricas de sucesso do MVP.

---

## 🎨 Passo 2: Idealização Gráfica (Wireframes)

Sem escrever nenhuma linha de código lógica de React Native, o segundo passo de extrema importância foi idealizar as 4 telas do fluxo do nosso usuário:

1. **Tela de Autenticação:** Como o idoso faz login.
2. **Tela Principal (Dashboard/Home):** A visualização amigável dos remédios do dia.
3. **Formulário de Cadastro:** A interface com campos de inputs simplificados para preencher "Nome", "Dosagem", etc.
4. **Resumo Semanal/Histórico:** Para apresentação a médicos de forma rápida.

**Resultados esperados:** Os designs e exportações de imagens de referência guardados na pasta pertinente, para que as Semanas Seguintes apenas "replicassem" o que foi idealizado.
