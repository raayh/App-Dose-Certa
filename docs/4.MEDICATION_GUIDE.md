This guide details how the medication enrollment screen was built, from the dynamic user interface down to the resilient Firestore write logic and safety controls.

## 🎨 Phase 1: Interface (The App’s Skeleton)

The medication form requires a clean layout capable of handling vertical overflow since users need to select types, frequency, and dates.

### 1. View & Scroll Container
We used a persistent fixed Header, a `ScrollView` for the content, and an Absolute View for safety locks.

**File:** `app/(tabs)/add.tsx`
```tsx

```

> [!TIP]
> **Coach Tip:** We set `tabBarStyle: { display: 'none' }` in the tab router settings specifically for this screen to give the user more breathing room to type and avoid accidental navigations.

## 🛡️ Phase 3: Premium Resilience & UX Controls

To transition from a basic MVP to a robust product, we implemented industry-level defense logic.

### 1. The Anti-Double-Action Shield
If `loading` is true, an absolute semi-transparent `View` is injected over everything to physically stop taps.
```tsx
{loading && (
  <View style={{ 
    position: 'absolute', top: 0, bottom: 0, 
    backgroundColor: 'rgba(255, 255, 255, 0.7)',
    zIndex: 999 
  }}>
     <ActivityIndicator />
  </View>
)}
```

### 2. Form State Clean-Up
React Native Tab screens stay alive in memory. To prevent outdated state leaks, we injected a memory wipe trigger right before leaving the screen:
```tsx
onPress: () => {
  resetForm(); // 1. Clean quietly
  router.replace('/(tabs)'); // 2. Swap route
}
```

## 🗃️ Phase 4: Hardened Firestore Persistency

### 1. Network Stability (Long Polling)
Mobile carriers or local Linux environments sometimes throttle web-channels. We hardcoded an infrastructure safeguard in `firebaseConfig.ts`:
```ts
export const db = initializeFirestore(app, {
  experimentalForceLongPolling: true, // Essential for React Native network stability
});
```

### 2. Atomic Safe Writing (`addDoc`)
Consistency between Backend Rules and Frontend keys is vital. We adhered stringently to the `user_id` snake_case mapping:
```tsx
await addDoc(collection(db, "medications"), {
  user_id: user.uid, // Crucial security pairing key
  name: name,
  times: times,
  startDate: startDate.toISOString(),
  created_at: new Date().toISOString()
});
```
